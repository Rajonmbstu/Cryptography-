Monoalphabetic Cipher

A substitution cipher that maps each letter of the plaintext alphabet to a unique ciphertext letter using a fixed substitution key. Simple but historically significant—used in Caesar ciphers, ROT13, and Atbash.

  • 26! possible keys
  • Auto-generate or enter custom key
  • Visual alphabet mapping table
Playfair Cipher

A digraph substitution cipher invented by Charles Wheatstone in 1854. Encrypts pairs of letters using a 5×5 keyword matrix. Used by the British military in WWI and WWII.

  • 5×5 Polybius matrix
  • I/J merging rule
  • Same-row, same-col, rectangle rules
Hill Cipher

A polygraphic cipher using linear algebra. Encrypts blocks of letters using matrix multiplication modulo 26. Invented by Lester S. Hill in 1929. First practical cipher using modern algebra.

  • 2×2 key matrix input
  • Invertibility validation mod 26
  • Matrix calculation walkthrough
